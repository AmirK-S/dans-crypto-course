<task type="auto">
  <name>Task 1: Install AI SDK and create transcript cleaning script</name>
  <files>package.json, scripts/clean-transcripts.mjs</files>
  <action>
  1. Run `npm install ai-sdk` in the project root. This installs the official AI SDK for Phase 3 summarization (used in Plan 03-02). Confirm it appears in package.json dependencies.

  2. Create `scripts/clean-transcripts.mjs` — a Node.js ESM script that:

  a) Reads all `.md` files from `output/transcripts/` recursively (all 12 section folders).

  b) For each file, extracts the body text AFTER the `## Transcript` or `## Content` heading. The metadata header (title, section, loom ID/source lines) MUST be preserved exactly — only the body text is cleaned.

  c) Applies these cleaning transformations to the body text (in this order):
     - Remove Loom VTT end-of-video artifacts:
       `/\bThanks\s+(?:for\s+)?watching[.!]?\s*/gi` → ''
       `/\bWe'll\s+see\s+you(?:\s+next)?\b[.!]?\s*/gi` → ''
       `/\bSee\s+you\s+(?:in\s+the\s+)?next(?:\s+one)?\b[.!]?\s*/gi` → ''
     - Remove filler sounds:
       `/\buh[hm]*\b\s*/gi` → ''
       `/\bu[hm]{2,}\b\s*/gi` → ''
       `/\bumm+\b\s*/gi` → ''
     - Remove VTT pause dots: `/\.\s+\.\s+\.\s*/g` → ' '
     - Remove Skool navigation boilerplate: `/Bullrun Millions Crypto Course.*?$/gms` → ''
     - Collapse multiple spaces to single: `/  +/g` → ' '
     - Trim the result

  d) Reconstructs the full file: metadata header + heading + cleaned body text.

  e) Writes the cleaned content back to the same file path (in-place update).

  f) Logs per-file: `[CLEAN] section/filename.md (removed N chars)` or `[SKIP] section/filename.md (no changes)`.

  g) At the end, prints a summary: total files processed, files modified, files skipped, total characters removed.

  IMPORTANT: Do NOT use `\buhh?\b` as a standalone pattern — it would match legitimate words. The patterns above are carefully scoped to match only filler sounds. The word-boundary `\b` prevents matching substrings of real words.

  IMPORTANT: YouTube stub files (containing "YouTube video embed") have minimal body text. The script should still process them but will likely skip them (no changes). Do not error on short files.
  </action>
  <verify>
    <automated>node scripts/clean-transcripts.mjs && echo "EXIT CODE: $?"</automated>
  </verify>

  - All 95 .md files are processed (modified or skipped)
  </done>
