# Introduction to the Course — Cheat Sheet

## Core Philosophy

- The course teaches **crypto investing from first principles** — not rules, but the reasoning behind the rules so you can adapt as markets evolve
- The instructor's primary edge is **fundamental analysis**: identifying coins before they 10-100x by evaluating project quality, not just price action
- Goal: build **3-5 "moon bags"** per bull run — high-conviction positions in fundamentally strong altcoins

## What This Course Covers

- Fundamental analysis (the core edge)
- Technical analysis (20% of the process — used to time entries/exits)
- Strategy building, mindset, tax/banking infrastructure, online income sources
- Airdrops and presales as additional income streams

## First Principles Approach

- Instead of memorizing rules, understand **why** each principle works
- Markets change; rules become obsolete; principles endure
- The instructor shares the actual reasoning used to make real investment decisions

## Key Expectations

- This is not a "get rich quick" scheme — it requires learning and patience
- Fundamental analysis requires more work than chart-watching, but produces more reliable results
- The course is designed for people serious about building wealth through crypto, not gamblers
